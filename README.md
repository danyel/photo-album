# Photo album

## Architecture
### Backend
#### Tech stack
nodejs
express
multer
crypto
fs
sharp
mime-types

#### Rest API

/api/photo-album
/api/thumb
/api/image
/api/upload

#### Configuration
dotEnv is used for the configuration.
So in your .env you need to define the PORT (default is 3000) of the application and the PHOTO_LIBRARY_LOCATION for storage and displaying the images

#### How to start
To install the dependencies
``
npm i
``
To run the application
``
npm run dev
``
